Frequently Asked Questions
Clarifications regarding scope and methodology.
Is this coverage?+

The diagnostic does not read or evaluate uploaded pages.

It operates through structured self-assessment scoring within a fixed framework.

Does this use AI to analyze my script?+

The diagnostic does not process script text.

Results are generated from weighted self-assessment inputs.

How is this different from using ChatGPT?+

ChatGPT generates open-ended feedback.

This diagnostic applies a fixed scoring model based on defined early-read elimination criteria.

It does not generate interpretation.

It classifies structural risk within defined bands.

Can I use the framework on multiple drafts?+

The Free Diagnostic allows a single assessment.

The Full Framework supports repeatable per-draft use.

Is this a guarantee of industry success?+

It functions as a structural filter prior to professional exposure.
